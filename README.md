# PBPERSONAL Smart Ledger

PBPERSONAL is a **Smart Personal Ledger Web App** to manage personal accounts, track money received, payments, and adjustments. It helps you easily see **balances per person**, export ledger to **PDF**, or share via **WhatsApp**.

---

## Features

- Add **Money Received**, **Payment**, or **Adjusted** entries
- Track transactions **per person**
- Auto-calculate **balance, total due, total payments**
- Export ledger as **PDF**
- Share ledger via **WhatsApp**
- Mobile-friendly & Desktop-ready
- Stores data in **local storage** (no backend needed)

---

## How to Use

1. Open `index.html` in your browser.
2. Add a person's **Name**, **Amount**, **Date**, **Transaction Type**, and optional **Note**.
3. Click **Add Entry** to save.
4. Click a person in the dashboard to see their **ledger view**.
5. Export or share ledger as needed.
